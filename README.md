# data-analysis-project1
End-to-End E-Commerce Sales &amp; Profitability Analysis using Python and Power BI. Built an executive dashboard with KPI tracking, revenue trends, customer insights, and growth analysis.

This project presents an end-to-end data analysis solution for an e-commerce retail business. The objective was to analyze transactional sales data, generate key performance indicators (KPIs), and build an executive-level interactive dashboard for business decision-making.

The project includes data cleaning, feature engineering, KPI creation, trend analysis, and dashboard visualization using Python and Power BI.
